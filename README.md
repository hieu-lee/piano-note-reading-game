# Đọc Nốt Nhạc Piano

A web-based game for practicing piano note reading on treble (khoá Son) and bass (khoá Fa) clefs, including sharps and flats.

## Features

- Random note generation on treble or bass clef staff
- Interactive 4-octave piano keyboard (C2–C6)
- Sharp and flat accidentals
- Real-time stats: accuracy and average response time
- Reference panels for C note positions
- Music reading rules reference

## Development

```bash
npm install
npm run dev
```

## Deploy to GitHub Pages

Push to `main` branch — GitHub Actions will build and deploy automatically.

## Tech Stack

Vite + React, SVG music notation, vanilla CSS.
